Projet 1 : Réseau Local d’Entreprise

Simulation d’un réseau informatique dans un environnement d’entreprise

🎯 Objectif

Mettre en place un réseau local permettant la communication entre plusieurs machines, avec un adressage IP structuré et des tests de connectivité.

🧱 Architecture du réseau

  • 1 routeur
  • 1 switch
  • 3 postes clients
  • Réseau : 192.168.1.0/24

🛠️ Outils utilisés

  • Packet Tracer / VirtualBox
  • Draw.io pour le schéma réseau
  • Commandes réseau (ping, ipconfig)

📡 Réalisation

J’ai configuré manuellement les adresses IP sur chaque machine afin de permettre la communication au sein du réseau local. Des tests de connectivité ont été effectués avec la commande ping.

📸 Résultats

Communication entre les machines validée avec succès.

📊 Conclusion

Ce projet m’a permis de comprendre les bases du fonctionnement d’un réseau local, l’importance de l’adressage IP et les outils de diagnostic réseau.
