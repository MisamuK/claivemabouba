🔐 Web Security Scanner (OWASP style)

Ce projet est un scanner de sécurité web qui analyse les headers HTTP d’un site afin de détecter des failles de sécurité courantes (OWASP basics).

🚀 Comment utiliser le projet

🟢 Version locale (chez toi)

Le projet fonctionne sur ton ordinateur en local. Ton PC agit comme serveur.

  • Lancer le serveur avec node server.js
  • Accéder depuis ton PC :
  • http://localhost:3000
  • Accéder depuis ton téléphone (WI-FI) :
  • http://192.168.x.x:3000

Exemple concret :

  • PC : http://localhost:3000
  • Téléphone (même Wi-Fi) : http://192.168.1.118:3000

⚠️ Important : les appareils doivent être sur le même réseau Wi-Fi.

🌍 Version en ligne (Internet)

Le scanner est hébergé sur un serveur en ligne (Render). Il peut être utilisé depuis n’importe où dans le monde.

Exemple :

https://web-security-scanner.onrender-omlr.com
🔐 Commencer le scan

👉 Clique sur le bouton pour lancer le scanner en ligne.

⚙️ Comment ça marche

  • Le serveur Node.js envoie une requête au site ciblé
  • Il analyse les headers HTTP (sécurité)
  • Il calcule un score de sécurité (0 à 100)
  • Il renvoie un rapport détaillé

💡 Local vs En ligne

Local : utilisé pour tester et développer

En ligne : accessible partout (portfolio / utilisateurs)

🔥 Fonctionnalités

  • 🔐 Analyse HTTPS
  • 🧱 Détection CSP
  • 🔐 Détection HSTS
  • 🍪 Analyse cookies sécurisés
  • 📊 Score de sécurité OWASP
