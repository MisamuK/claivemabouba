💬 Chat Temps Réel (Node.js + Socket.io)

Ce projet est un chat en temps réel permettant à plusieurs utilisateurs de discuter instantanément grâce à un serveur Node.js et WebSocket (Socket.io).

🚀 Comment utiliser le projet

🟢 Version locale (chez toi)

  • Lancer le serveur avec node server.js
  • Ouvrir sur ton PC : http://localhost:3000
  • Ouvrir sur ton téléphone (même Wi-Fi) :
  • http://192.168.x.x:3000

Exemple concret :

  • PC : http://localhost:3000
  • Téléphone (même Wi-Fi) : http://192.168.1.118:3000

⚠️ Important : le PC et le téléphone doivent être sur le même réseau Wi-Fi.

🌍 Version en ligne (Internet)

Le chat peut aussi être hébergé sur Internet. Ici nous avons utilisé Render qui est gratuit pour ce type de déploiement. Dans ce cas, n’importe qui peut s’y connecter depuis le monde entier.

Exemple :

https://chat-reseau.onrender.com
💬 Commencer le chat

👉 Clique sur le bouton pour accéder directement au chat en ligne.

⚙️ Comment ça marche

  • Le serveur Node.js gère les connexions des utilisateurs
  • Socket.io permet la communication en temps réel
  • Chaque message est envoyé instantanément à tous les utilisateurs connectés

💡 Différence entre local et en ligne

Local : utilisé pour tester sur ton réseau Wi-Fi personnel

En ligne : accessible partout sur Internet 24h/24

🔥 Fonctionnalités

  • 💬 Chat en temps réel
  • 👤 Gestion des utilisateurs
  • 🟢 Notifications connexion / déconnexion
  • 📱 Compatible mobile et PC
